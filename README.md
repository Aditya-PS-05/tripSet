## Project Setup Guide

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Git

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/tripset.git
cd tripset
```
### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
